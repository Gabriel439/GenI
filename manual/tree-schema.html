Utilities

A TAG node may have a category. In the core GenI algorithm, there is nothing which distinguishes the category from any other attributes. But for some other uses, such as checking if it is a result or for display purposes, we do treat this attribute differently. We take here the convention that the category of a node is associated to the attribute “cat”.

A TAG node might also have a lexeme. If we are lucky, this is explicitly set in the glexeme field of the node. Otherwise, we try to guess it from a list of distinguished attributes (in order of preference).
